# Loja de Flores

Projeto de uma loja virtual desenvolvida em HTML, CSS e JavaScript.

## Funcionalidades
- Exibição de produtos
- Página de detalhes
- Carrinho de compras
- Tela de loading

## Tecnologias
- HTML5
- CSS3
- JavaScript
- bootstrap

## Como executar
1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
